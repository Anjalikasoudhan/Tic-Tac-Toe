# Tic-Tac-Toe

A classic, simple Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Features Let's List
- 3x3 play grid
- Interactive UI to mark squares

## Usage
Simply open `index.html` in your web browser.
